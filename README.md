# Verified Open Source Projects

A repository for verified developers to share open-source projects.

## Purpose
This repo is a collection of open-source projects submitted by verified developers.  
Anyone can view, use, and contribute — **commercial use is not allowed** unless explicitly permitted.

## How to Contribute
1. Fork this repository
2. Add your project in a new folder with:
   - Project name
   - Description
   - `README.md`
   - Source code
3. Open a pull request

## Rules
- You must be a **verified developer** (proof required)
- Projects must be open source
- No commercial use without permission
- Give credit when using others' work

## License
